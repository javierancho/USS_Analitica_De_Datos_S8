# USS_Analitica_De_Datos_S8
#Taller sumativo S8 de Analítica y Ciencia de Datos: EDA y modelos predictivos sobre Boston Housing.

## Descripción

Este trabajo tiene como objetivo analizar y predecir el valor mediano de las viviendas (MEDV) utilizando el dataset Boston Housing.

Se realizó un análisis exploratorio de datos (EDA), se aplicaron transformaciones y se implementaron distintos modelos predictivos para comparar su desempeño.

---

## Contenido

- Inspección y limpieza de datos
- Análisis exploratorio (EDA)
- Análisis de valores atípicos
- Análisis de correlaciones
- Modelos:
  - Regresión Lineal
  - Redes Neuronales
  - Árbol de Decisión
  - XGBoost
- Evaluación (RMSE y R²)
- Comparación de modelos

---

## Resultados

- XGBoost obtuvo el mejor desempeño
- Variables más influyentes:
  - LSTAT
  - RM
- Se identificó posible sobreajuste en modelos complejos

---

## Tecnologías

Python, pandas, numpy, scikit-learn, keras, xgboost, matplotlib, seaborn

---

## Conclusión

El modelo XGBoost logra la mejor precisión, mientras que la regresión lineal ofrece mayor interpretabilidad.
